# MaxKonvert — Правила и выплаты

## Правила использования (запрещено)

- Использовать номера и префиксы MaxKonvert на других сайтах.
- Сливать трафик, добытый с помощью вредоносных программ.
- Регистрироваться под собственной реферальной ссылкой.
- Самостоятельно подписывать номера абонентов.
- Регистрировать несколько аккаунтов (мультиаккаунтинг).

Администрация оставляет за собой право отказать в участии любому сайту без объяснения причин.

## Расписание выплат

- **Периодичность:** каждый четверг.
- **Период:** прошлая неделя (понедельник — воскресенье).
- **Минимальная сумма:** отсутствует (по состоянию на дату проверки).

## Способы вывода

- WebMoney (WMR) — подтверждено на сайте.
- По другим вариантам вывода актуальные условия нужно уточнять через поддержку.

## Быстрый ответ: какие есть способы вывода

Основной подтверждённый способ вывода: **WebMoney (WMR)**.
По остальным вариантам выводов нужно уточнять актуальные условия через поддержку.
