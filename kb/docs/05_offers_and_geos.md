# MaxKonvert — Офферы и GEO

## Доступные форматы монетизации

1. **МТ-подписки** — подписки на платный контент с регулярными ребиллами.
2. **WAP-клик** — монетизация мобильного трафика через клики по WAP-ссылкам.
3. **Псевдо-подписки** — пользователь вводит номер, отвечает на бесплатное SMS, после чего происходит списание.
4. **Премиум-SMS** — оплата услуги через отправку SMS на короткий номер.
5. **Платники с голосовым сопровождением** — лендинги с голосовой инструкцией по регистрации / подписке.
6. **Push-подписки** — временно доступны с корректировкой цены выкупа (см. новости).

## Условия отчислений

- Базовая ставка: от 80% до 95% от дохода.
- Для крупных партнёров с большим объёмом трафика возможны индивидуальные повышенные ставки после обсуждения с менеджером.

## Настройки оплат

Система предлагает автоматически выставить самые выгодные настройки, но партнёр всегда может изменить их вручную.
